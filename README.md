# 🚀 Premium Portfolio Website

Современный, профессиональный сайт-портфолио с 3D-эффектами, плавными анимациями и полной адаптивностью.

## ✨ Особенности

- 🎨 **Современный дизайн** - Pixel-perfect UI с продуманной типографикой
- 🎭 **3D-эффекты** - Интерактивные 3D-элементы с использованием Three.js и React Three Fiber
- 🌊 **Плавные анимации** - Бесшовные переходы и микро-взаимодействия с Framer Motion
- 📱 **100% адаптивность** - Mobile-First подход, идеальный вид на всех устройствах
- 🌍 **Мультиязычность** - Поддержка русского, узбекского и английского языков
- ⚡ **Высокая производительность** - Оптимизированный код и быстрая загрузка
- 🎯 **SEO-оптимизация** - Правильная структура и метатеги

## 🛠️ Технологии

- **React 18** - Современная библиотека для создания UI
- **Vite** - Быстрый сборщик и dev-сервер
- **Tailwind CSS** - Utility-first CSS фреймворк
- **Framer Motion** - Библиотека для анимаций
- **Three.js** - 3D-графика в браузере
- **React Three Fiber** - React-рендерер для Three.js
- **Lucide React** - Красивые иконки

## 📦 Установка и запуск

### Предварительные требования

- Node.js версии 16 или выше
- npm или yarn

### Шаги установки

1. **Клонируйте репозиторий или распакуйте архив**

2. **Установите зависимости**

```bash
npm install
```

или

```bash
yarn install
```

3. **Запустите dev-сервер**

```bash
npm run dev
```

или

```bash
yarn dev
```

4. **Откройте браузер**

Приложение автоматически откроется по адресу `http://localhost:3000`

## 🎨 Кастомизация

### Изменение языка

Сайт поддерживает три языка:
- 🇷🇺 **Русский** (по умолчанию)
- 🇺🇿 **O'zbekcha**
- 🇬🇧 **English**

Переключатель языка находится в правом верхнем углу навигационной панели. Выбранный язык сохраняется в localStorage браузера.

**Добавление нового языка:**

1. Откройте `src/translations/translations.js`
2. Добавьте новый объект языка (например, для французского):

```javascript
fr: {
  nav: {
    home: 'Accueil',
    about: 'À propos',
    // ... остальные переводы
  },
  // ... все секции
}
```

3. Добавьте язык в `src/components/Navbar.jsx`:

```javascript
const languages = [
  { code: 'ru', name: 'Русский', flag: '🇷🇺' },
  { code: 'uz', name: 'O\'zbekcha', flag: '🇺🇿' },
  { code: 'en', name: 'English', flag: '🇬🇧' },
  { code: 'fr', name: 'Français', flag: '🇫🇷' }, // новый язык
]
```

### Изменение цветовой схемы

Откройте `tailwind.config.js` и измените цвета в секции `theme.extend.colors`:

```javascript
colors: {
  primary: {
    // Ваши цвета
  }
}
```

### Изменение контента

Все тексты находятся в файле `src/translations/translations.js`. Отредактируйте нужные секции для каждого языка:

- **nav** - Навигационное меню
- **hero** - Главная секция
- **about** - Информация о вас
- **skills** - Ваши навыки
- **projects** - Ваши проекты
- **contact** - Контактная информация
- **footer** - Подвал сайта

### Замена изображений проектов

В `src/components/Projects.jsx` замените URL изображений на свои:

```javascript
image: 'ваш-url-изображения'
```

## 🌍 Мультиязычность

Система переводов построена на React Context API и поддерживает:

- Автоматическое сохранение выбранного языка
- Мгновенное переключение без перезагрузки страницы
- Легкое добавление новых языков
- Полный перевод всех элементов интерфейса

**Как это работает:**

1. `LanguageContext` управляет текущим языком
2. `translations.js` содержит все переводы
3. Каждый компонент использует `useLanguage()` hook
4. Переводы применяются через `t.section.key`
